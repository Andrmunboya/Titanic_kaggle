# 🚢 Titanic: Machine Learning from Disaster | End-to-End Pipeline

Решение знаменитой задачи Kaggle по прогнозированию выживаемости пассажиров Титаника. Проект прошел путь от базового анализа данных (EDA) до построения production-ready API для инференса моделей.

---

## 🎯 Описание проекта

Цель проекта — предсказать, какие пассажиры выжили во время крушения Титаника, используя методы машинного обучения и данные о пассажирах (возраст, пол, класс билета и т.д.). 

В рамках этого репозитория я объединил несколько подходов к решению задачи:
1. **Классический Machine Learning** с использованием алгоритма Random Forest.
2. **Продвинутый градиентный бустинг** с помощью библиотеки CatBoost.
3. **Деплой модели** — создание легковесного API для получения предсказаний в реальном времени.

## 🚀 Как запустить проект локально

### 1. Клонирование репозитория
```bash
git clone https://github.com
cd название-твоего-репозитория
```

### 2. Установка зависимостей
Рекомендуется использовать виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Для Linux/macOS
# или venv\Scripts\activate  # Для Windows

pip install -r api/requirements.txt
```

### 3. Запуск API
Перейдите в папку с API и запустите сервер:
```bash
cd api
python app.py
```
После этого API будет доступно по адресу `http://127.0.0.1:5000`.
